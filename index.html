이번 달 소재 현황 요약
캘린더 상단에 이번 달 소재 총 개수와 분류별 수량이 자동으로 표시돼요. 월간 보고할 때 바로 활용할 수 있어요.
🔍
소재 상세보기
캘린더의 소재 박스를 클릭하면 상세보기가 먼저 열려요. 분류, 기간, 플랫폼 아이콘, 메모를 한눈에 확인할 수 있어요. 수정이 필요할 때만 수정 버튼을 누르면 돼요.
📋
소재 복사해서 추가
상세보기에서 복사해서 추가 버튼을 누르면 해당 소재의 내용이 그대로 복사된 추가 창이 열려요. 날짜와 소재명만 바꿔서 반복 소재를 빠르게 등록할 수 있어요.
📅
소재 추가
우측 상단 + 소재 추가 버튼 또는 날짜 칸의 + 버튼으로 소재를 추가할 수 있어요.
🗂️
분류 필터
헤더의 분류 뱃지를 클릭하면 켜고 끌 수 있어요. 특정 권역 소재만 보고 싶을 때 나머지를 꺼서 필터링하세요. 단, 마지막 하나는 끌 수 없어요.
🖱️
드래그 앤 드롭으로 일정 이동
소재 박스를 잡아서 다른 날짜로 드래그하면 일정이 이동돼요. 장기 일정은 기간을 유지한 채 통째로 이동돼요.
📅
장기 일정 (주간 이벤트)
시작일과 종료일을 다르게 설정하면 캘린더에 가로로 쭉 이어지는 바 형태로 표시돼요.
📱
플랫폼 아이콘
소재 추가/수정 시 플랫폼을 아이콘 버튼으로 중복 선택할 수 있어요. X, Facebook, Instagram, Youtube, Discord를 지원해요.
🔄
실시간 동기화
팀원이 소재를 추가/수정/삭제하면 다른 사람 화면에도 즉시 자동 반영돼요.
🏠
오늘로 돌아가기
← 오늘 → 버튼 중 오늘을 누르면 이번 달로 바로 돌아와요.
캘린더 사용 간 문제 발생 시 이시현_NC로 연락 주세요
